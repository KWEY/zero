# KWE

## REPL Environment

-   [TypeScript](https://www.typescriptlang.org/play/)

## How to develop and build

-   克隆当前的项目

```
$ git clone git@https://github.com/kwey/zero.git
```

-   在项目的根目录初始化依赖项

```
$ npm install
```

-   开发方式壹

```
$ npm run dev
# 端口8080
```

-   开发方式贰

```
$ npm run debug
```

-   单元测试

```
$ npm run jest
```

-   e2e 测试

```
先开启服务
$ npm run dev

在新窗口运行e2e
$ npm run e2e
```

-   打包上线

```
$ npm run build
```
